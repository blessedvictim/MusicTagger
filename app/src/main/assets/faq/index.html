This application provides you to edit the metadata of your music files. That mean edit track name, album, artist, etc., which are used by media players and other programs, as well as hardware players to display information about file and automatically organize your audio collections.
Smart search is function to look up correct metadata for track in automatically mode, which saves you from manual tag editing. To find out how you can use smart search, check the help. Pay attention, this function is only applicable to files that have a minimum audio track length of 120 second.
Smart search works by taking a digital fingerprint from the track and searching for it metadata in the public services, which may not contain data on this tracks (especially if the performer is unpopular or the song is too new). You may also meet problems if the audio track has been modified by some software.
To quickly navigate in music collection, you can groups tracks by artist, album or show all files at once.
With a long tap, you can turn on selection mode. Here you can choose few tracks at once and edit tags in multiply mode (Album mode).
For more information check out help.
At first run, we asked you for permission to write in memory and choose sd-card path root. You may have choose the wrong root path of your SD-card. You can always check it out and change sd-card path in application settings.
