Данное приложение позволяет редактировать метаданные ваших музыкальных файлов, то есть изменять данные о названии трека, альбома, имени исполнителя и т. д., которые используются медиаплеерами и другими программами, а также аппаратными проигрывателями для отображения информации о файле и автоматического упорядочивания аудиоколлекции.
"Умный" поиск - функция находящая правильные метаданные для трека в автоматическом режиме, избавляя вас от ручного редактирования тегов. Чтобы узнать как можно использовать умный поиск, загляните в справку.Обратите ванимание что данная функция применима только к файлах у которых длина аудиодорожки больше 120 секунд.
"Умный" поиск работает посредством снятия с трека цифрового отпечатка и поиска по нему тегов в отрытых источниках, которые могут содержать данные не о всех треках(особенно если исполнитель непопулярный или песня слишком новая). Также проблемы могут возникнуть если звуковая дорожка была изменена с помощью какого либо ПО.
Для быстрой навигации по музыкальной коллекции вы можете группировать файлы по исполнителю,по альбому или показывать все файлы сразу.
Долгим нажатием вы можете включить режим выделения, с помощью которого вы можете редактировать теги в режиме альбомов!?
